# gotris

tetris written in go

## Controls:

| #   | Key   | Action     |
| --- | ----- | ---------- |
| 1   | S     | Start      |
| 2   | Q     | Quit       |
| 3   | ←     | Move left  |
| 4   | →     | Move right |
| 5   | ↑     | Rotate     |
| 6   | ↓     | Soft drop  |
| 7   | Space | Hard drop  |
